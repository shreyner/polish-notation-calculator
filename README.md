# Польская нотация
Конвертация инфиксной записи в постфиксную и вычисление значения.

Для запуска:
- Поставить NodeJS v16 
- Запустить программу
```bash
$ npm run start
```

Формат записили:

Операторы и числа разделить пробелом.

Пример:
```bash
Infix > 3 * 3 / ( 7 + 1 )
Infix to postfix: 3 3 * 7 1 + /
Result calculation: 1.125
```
